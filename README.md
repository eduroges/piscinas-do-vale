# Piscinas do Vale

Site institucional da Piscinas do Vale, especialista em projetos e construção de piscinas de luxo e alto padrão.

## Tecnologias Utilizadas

- React
- Vite
- Tailwind CSS
- Shadcn UI
- Lucide React
- Framer Motion

## Como executar o projeto

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Execute o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
